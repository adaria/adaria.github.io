SVT-AV1 command line changelog

from 02/01/2020 --------------------

SvtAv1EncApp -w <width> -h <height> -scd 0 -lp <lp> -enc-mode 0 -q <qp> -intra-period <kf interval> -n <number of frames> -bit-depth <bit depth> -fps-num <fps num> -fps-denom <fps denom> -i <input YUV> -b <output bitstream>

<lp> = 24 for 8bit sources, 12 for 10bit sources

<kf interval> = 4 seconds (e.g. 120 frames for 30 fps, 96 frames for 24 fps)

<qp> = {20, 32, 43, 55, 59}
