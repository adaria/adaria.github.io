SVT-AV1 command line changelog

from 01/01/2020 --------------------

Pass 1:

SvtAv1EncApp -w <width> -h <height> -scd 0 -lp 12 -enc-mode 5 -enc-mode-2p 0 -output-stat-file two_pass_stats -q <qp> -intra-period <kf interval> -n <number of frames> -bit-depth <bit depth> -fps-num <fps num> -fps-denom <fps denom> -i <input YUV> -b <output bitstream>

Pass 2:

SvtAv1EncApp -w <width> -h <height> -scd 0 -lp 12 -enc-mode 0 -input-stat-file two_pass_stats -q <qp> -intra-period <kf interval> -n <number of frames> -bit-depth <bit depth> -fps-num <fps num> -fps-denom <fps denom> -i <input YUV> -b <output bitstream>

<kf interval> = 4 seconds (e.g. 120 frames for 30 fps, 96 frames for 24 fps)

<qp> = {20, 32, 43, 55, 59}
